Features to Provide Foccus

  • Needles: Search the input to hooked functions for important strings or regex.
  • Blacklist: Don't show results if the input contains a string or regex from the blacklist.
  • Targets: Hook functions only if they are on a target page, reduce noise from other framed pages.
  • Functions: Pick almost any native JavaScript function or setter to be hooked
  • Enable/Disable: Disable features you are not using to reduce noise or improve speed.
  • Auto Open: Change which console groups are automaticly open depending on your interests.

    • Changes to configuration requrie a page load
